プログラム利用者の方へ

1.　サービス内容
  　希望の年月日と地点に関する1日単位の気温グラフを閲覧です。
　　※　地点は「岐阜県」、「三重県」、「静岡県」、「愛知県」内の各地点です。

2.　各プログラムの概要
　1 )　「Server_App3」フォルダ内の「Storage.java」を実行。
	指定フォルダにあるCSVファイルのデータが、データベースに加工されて保存されます。

　2 )　
     ( 1 )　「Cl_App2」フォルダ内の「WebContent」フォルダ内の「Portal.html」を実行。
     ( 2 )　『年月日』と『都道府県』と『地点』を選択します。
　　 ( 3 )　「servlet」フォルダ内の「Cl_servlet.java」と「mybeans」フォルダの「ClBean.java」を通じて、
　　　　　　　「WebContent」フォルダ内の「WEB-INF」フォルダ内の「Result.jsp」に選択条件に合致する一日の気温グラフが表示されます。
　 
3. 準備
　1 )気象庁のホームページから必要な地点のCSVデータをパソコンの任意フォルダに出力します。
     ※ 毎月同じ日にちに、「期間」を「最近1ヶ月」にして気象庁からデータ取得すれば、年月日時は連続します。
　2 )　アプリケーションサーバ( Apache Tomcat)を設定してください( ※1 )。
　3 )　各プロジェクトのプロパティを開き、「Javaのビルド・パス」へ進み、外部JARの追加(X)...」ボタンを押し、
　　　「SQLite-JDBCドライバ」( ※2 )を選択、「開く」ボタンを押してください。
  4 )　各プログラムに関する個別設定 : 
　　	( 1 )　「Server_App3」 : 
		①　「Storage.java」内の38行目「get_file =("...")」の「...」に、
		　気象庁のホームページからダウンロードしたCSVファイルまでのパスを指定します( ※3 )。
		②　「Storage.java」内の41行目「db_url =("...")」の「...」に、
		　データベースファイル名(.db)を記述してください( ※4 )。

	( 2 )　「Cl_App2」 : 
　　		①　「WEB-INF」フォルダの「lib」フォルダに「SQLite-JDBCドライバ」( ※1 )をコピーしてください。
　　		②　「Cl_servlet.java」内の51行目「db_url =("...")」の「...」に
		   データベースファイル名(.db)を記述してください( ※4 )。

4.　Javaプログラムの使い方
　1 )　　「Server_App3」 : 
	( 1 )　「Storage.java」を実行してください。

	( 2 )　アプリ内の「実行」ボタンを押すと、CSVファイルのデータをデータベースに取り込みます。

	( 3 )　データ取り込みが終わると、進捗状況欄にメッセージが記載されます。

	( 4 )　データを抽出されたCSVデータは自動的に削除されます。
	

　2 )　　　「Cl_App2」 : 
	( 1 )　「WebContent」フォルダの「Portal.html」を実行してください。

	( 2 )　「年月日」、「都道府県」、「地点名」を選択してください。選択された都道府県名に応じて、
		各都道府県のどの地点名リストを用いるかプログラムが判断します。

	( 3 )　条件に適合するグラフが表示されます。


＜　注釈　＞
※ 1.　「Cl_App2」と「Server_App3」の初期設定として、「Tomcat v8.5サーバー」が登録されています。

※ 2.　「Cl_App2」と「Server_App3」の初期設定として、「sqlite-jdbc-3.30.1」が登録されています。

※ 3.　基本的に気象庁のホームページからダウンロードしたCSVファイルは、パソコンの「ダウンロード」フォルダに
	「data.csv」として保存されます。初期設定として、「C:/Users/~/Downloads/data.csv」が登録されています。

※ 4. 初期設定として、「jdbc:sqlite:C:/Users/~/Temperature/SQLite/testDB6.db」が登録されています。
